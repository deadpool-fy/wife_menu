# 家庭菜单微信小程序

一个专为家庭设计的微信小程序，帮助妻子选择每天的饮食，并可以一键分享给丈夫。

## 功能特点

### 🍽️ 主要功能
- **菜品浏览**: 浏览各种家常菜，包含图片、菜名、分类等信息
- **详情查看**: 点击菜品可查看详细的制作方法、食材清单、制作步骤
- **分类筛选**: 根据荤素、荤素搭配等分类浏览菜品
- **菜单选择**: 选择多道菜品组成当天的菜单
- **微信分享**: 一键将选好的菜单分享给丈夫
- **菜品评价**: 对每道菜进行评分和评价，评价内容自动发送给丈夫

### 🎨 界面设计
- 现代化UI设计，色彩搭配温馨
- 响应式布局，适配不同屏幕尺寸
- 直观的操作流程，用户体验友好

### 📱 页面结构
- **首页**: 今日推荐菜品，已选菜品展示
- **分类页**: 按分类浏览所有菜品
- **详情页**: 菜品详细信息，制作方法
- **评价页**: 对已选菜品进行评分和评价

## 使用说明

### 1. 浏览菜品
- 在首页查看今日推荐菜品
- 点击"分类浏览"按分类查看所有菜品
- 点击菜品图片或名称查看详情

### 2. 选择菜单
- 点击菜品上的"选择"按钮添加到菜单
- 在首页查看已选择的菜品
- 可以移除不需要的菜品

### 3. 分享菜单
- 选择好菜品后，点击"发送给老公"
- 系统会自动生成菜单内容
- 可以分享到微信或直接发送消息

### 4. 评价菜品
- 发送菜单后会自动跳转到评价页面
- 对每道菜进行1-5星评分
- 可以添加文字评价
- 评价内容会自动发送给丈夫

## 数据说明

### 菜品数据结构
```javascript
{
  id: 1,                    // 菜品ID
  name: '宫保鸡丁',          // 菜品名称
  category: '荤菜',          // 菜品分类
  categoryType: 'meat',     // 分类类型
  difficulty: '中等',        // 难度等级
  cookingTime: '30分钟',     // 制作时间
  servings: 2,              // 适合人数
  calories: '320卡',        // 热量
  image: '图片URL',          // 菜品图片
  description: '菜品描述',   // 菜品介绍
  ingredients: [...],       // 食材清单
  steps: [...],             // 制作步骤
  tips: '制作小贴士'        // 小贴士
}
```

### 分类类型
- `meat`: 荤菜
- `vegetable`: 素菜
- `mixed`: 荤素搭配
- `soup`: 汤类
- `dessert`: 甜品

## 部署说明

### 1. 开发环境
- 安装微信开发者工具
- 导入项目代码
- 配置小程序AppID

### 2. 生产环境
- 在微信公众平台注册小程序
- 配置服务器域名
- 上传代码并提交审核

### 3. 数据更新
- 每月1号自动检查菜单更新
- 支持从服务器获取最新菜品数据
- 可以手动触发数据更新
