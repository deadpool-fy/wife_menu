# 🛠️ 后台管理系统使用指南

## 🎯 功能概述

现在您的小程序已经集成了完整的后台管理系统，可以：

1. **手动录入菜谱** - 添加详细的菜谱信息
2. **菜谱管理** - 查看、编辑、删除菜谱
3. **数据管理** - 初始化、清理数据
4. **系统监控** - 查看统计信息和日志

## 🚀 使用方法

### 1. 进入后台管理

1. **打开小程序**
2. **点击「管理」按钮**（在首页右上角）
3. **进入后台管理页面**

### 2. 菜谱管理

#### 添加菜谱
1. **点击「添加菜谱」按钮**
2. **填写菜谱信息**：
   - 菜谱名称（必填）
   - 菜谱描述（必填）
   - 上传图片（可选）
   - 选择分类（必填）
   - 设置难度、时间、人数
   - 添加食材清单
   - 编写制作步骤
   - 添加小贴士和标签
3. **点击「提交菜谱」**

#### 管理菜谱
1. **点击「菜谱管理」按钮**
2. **查看菜谱列表**：
   - 浏览所有菜谱
   - 使用筛选器（分类、难度、搜索）
   - 查看菜谱详情
3. **编辑菜谱**：
   - 点击「编辑」按钮
   - 修改菜谱信息
   - 保存更改
4. **删除菜谱**：
   - 点击「删除」按钮
   - 确认删除操作

### 3. 系统管理

#### 测试连接
- **点击「测试连接」** - 检查云函数连接状态
- **查看连接结果** - 确认系统正常运行

#### 初始化数据
- **点击「初始化数据」** - 添加示例菜谱和分类
- **确认操作** - 系统会添加基础数据

### 4. 数据清理

#### 清空所有数据
- **点击「清空所有数据」** - 删除所有菜谱和分类
- **确认操作** - 此操作不可恢复

#### 清空菜谱
- **点击「清空菜谱」** - 只删除菜谱数据
- **保留分类** - 分类数据不受影响

#### 清空分类
- **点击「清空分类」** - 只删除分类数据
- **保留菜谱** - 菜谱数据不受影响

## 📊 功能详解

### 菜谱录入功能

#### 基本信息
- **菜谱名称** - 必填，最多50个字符
- **菜谱描述** - 必填，最多200个字符
- **菜谱图片** - 可选，支持相册和拍照

#### 分类信息
- **菜谱分类** - 必选，包含：荤菜、素菜、汤类、甜品、主食、荤素搭配
- **难度等级** - 可选：简单、中等、困难
- **制作时间** - 可选：10分钟、30分钟、1小时、1.5小时、2小时
- **适合人数** - 可选：1人、2人、3人、4人、5人以上
- **卡路里** - 可选，数字输入

#### 食材清单
- **动态添加** - 点击「添加食材」按钮
- **食材名称** - 输入食材名称
- **用量** - 输入具体用量
- **删除食材** - 点击「×」按钮

#### 制作步骤
- **动态添加** - 点击「添加步骤」按钮
- **步骤描述** - 输入详细制作步骤
- **删除步骤** - 点击「×」按钮

#### 小贴士和标签
- **制作小贴士** - 分享制作心得
- **标签管理** - 输入标签后按回车添加

### 菜谱管理功能

#### 列表展示
- **菜谱卡片** - 显示图片、标题、描述
- **元数据** - 显示分类、难度、时间
- **标签展示** - 显示相关标签
- **操作按钮** - 编辑、删除按钮

#### 筛选功能
- **分类筛选** - 按菜谱分类筛选
- **难度筛选** - 按难度等级筛选
- **搜索功能** - 按菜谱名称搜索

#### 分页功能
- **自动分页** - 每页显示20个菜谱
- **加载更多** - 点击「加载更多」按钮
- **分页信息** - 显示当前页数和总页数

### 系统管理功能

#### 统计信息
- **总菜谱数** - 显示菜谱总数
- **分类数量** - 显示分类总数
- **最近添加** - 显示最近添加的菜谱数

#### 系统日志
- **实时日志** - 显示系统操作日志
- **日志类型** - 成功、错误、警告、信息
- **时间戳** - 显示操作时间
- **刷新日志** - 点击刷新按钮

## ⚠️ 注意事项

### 使用建议
- **定期备份** - 定期导出重要数据
- **分类管理** - 合理使用分类功能
