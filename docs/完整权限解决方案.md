# 微信小程序订阅消息权限完整解决方案

## 权限错误 -604101 分析

### 错误含义
`-604101` 表示云函数没有权限调用订阅消息API，这是微信小程序最新的权限控制机制。

## 根本解决方案

### 第一步：令牌权限配置（关键步骤）

这是解决权限问题的**核心步骤**：

1. **进入云开发控制台**：
   - 在微信开发者工具中点击 **"云开发"**
   - 或访问 https://console.cloud.weixin.qq.com/

2. **配置令牌权限**：
   ```
   设置 → 权限设置 → 令牌权限配置
   ↓
   在输入框中添加：/wxa/openapi
   ↓
   点击"保存"按钮
   ```

3. **重新部署云函数**：
   - 右键 `cloudfunctions/sendMessage` → "上传并部署"
   - 等待部署完成

#### 1. 用户授权检查
```javascript
// utils/authorization.js 确保授权逻辑正确
function requestSubscribeMessage(templateId) {
  return new Promise((resolve, reject) => {
    wx.requestSubscribeMessage({
      tmplIds: [templateId],
      success: (res) => {
        // 检查授权状态
        console.log('订阅消息授权结果:', res)
        if (res[templateId] === 'accept') {
          resolve({ success: true })
        } else {
          reject({ success: false, message: '用户拒绝接收消息' })
        }
      },
      fail: reject
    })
  })
}
```
