# 权限错误 -604101 解决方案

## 错误信息
```
errCode: -604101 function has no permission to call this API | errMsg: system error: error code: -604101
```

## 问题原因
错误 `-604101` 表示云函数没有调用订阅消息 API 的权限。

## 解决步骤

### 第一步：检查云开发权限设置

1. **打开云开发控制台**：
   - 在微信开发者工具中点击 **"云开发"**
   - 或访问 https://console.cloud.weixin.qq.com/

2. **进入权限设置**：
   - 点击 **"设置"** → **"权限设置"**
   - 查看 **"微信API调用权限"**

3. **开启订阅消息权限**：
   - 找到 **"订阅消息"** 相关权限
   - 点击 **"开通"** 或 **"开启"**
   - 如果没有该选项，需要先申请开通

### 第二步：检查小程序开放平台权限

1. **微信公众平台设置**：
   - 访问 https://mp.weixin.qq.com/
   - 使用管理员账号登录

2. **开通订阅消息功能**：
   - 进入 **"功能"** → **"订阅消息"**
   - 确认功能已开通
   - 检查模板状态是否为 **"已审核通过"**

### 第三步：检查云函数部署

1. **重新部署云函数**：
   - 右键 `cloudfunctions/sendMessage` 
   - 选择 **"上传并部署：云端安装依赖"**
   - 确保包含 `wx-server-sdk` 依赖

2. **检查环境ID**：
   - 确认 `app.js` 中的环境ID正确
   - 与云开发控制台中的环境ID一致

### 第四步：临时解决方案（降级处理）

如果权限暂时无法开通，可以先使用降级方案：

```javascript
// cloudfunctions/sendMessage/index.js 临时版本
exports.main = async (event, context) => {
  console.log('云函数执行，参数:', event)
  
  const { message, targetUserId, templateId } = event
  
  // 构建消息数据
  const messageData = {
    thing1: { value: '已生成待确认菜谱' },
    time2: { value: new Date().toLocaleString('zh-CN') },
    thing4: { value: message.length > 20 ? message.substring(0, 20) + '...' : message }
  }
  
  console.log('准备的消息数据:', messageData)
  
  // 临时返回成功，避免权限错误
  return {
    success: true,
    message: '消息已准备（需要手动转发，权限问题待解决）',
    data: messageData,
    note: '因权限限制，请手动复制消息内容转发'
  }
}
```

### 第五步：完整解决方案

#### 方案1：申请订阅消息权限

1. **在云开发控制台**：
   - 查看是否有权限申请入口
   - 申请开通订阅消息权限

2. **在微信公众平台**：
   - 确认小程序已通过审核
   - 申请订阅消息功能权限

#### 方案2：使用统一服务消息

如果能申请到统一服务消息权限：

```javascript
// 使用统一服务消息（需要特殊权限）
const result = await cloud.openapi.uniformMessage.send({
  touser: targetUserId,
  weapp_template_msg: {
    template_id: templateId,
    page: 'pages/index/index',
    data: messageData
  }
})
```

#### 方案3：继续使用降级方案

保持当前的降级机制：
1. 显示完整菜单内容
2. 复制到剪贴板
3. 手动转发给丈夫

## 验证步骤

### 1. 检查云函数日志
- 部署修改后的云函数
- 测试消息发送
- 查看是否有详细日志输出

### 2. 测试降级功能
- 选择菜品
- 点击发送
- 确认复制到剪贴板功能正常

### 3. 检查小程序权限
- 在小程序中重新授权
- 清除小程序数据后重新登录

## 关键检查点

1. **云开发权限**：云开发控制台 → 设置 → 权限设置
2. **小程序权限**：微信公众平台 → 功能 → 订阅消息
3. **模板状态**：确认模板已审核通过
4. **环境配置**：确认环境ID正确

## 下一步

1. **立即检查权限设置**
2. **如果权限充足**：正常使用订阅消息
3. **如果权限不足**：使用降级方案
4. **长期方案**：申请必要的API权限

## 临时使用建议

在权限问题解决前：
1. **使用复制功能**手动转发消息
2. **记录云函数日志**用于排查
3. **申请相关权限**用于长期使用
