# 后台入口与权限

## 现在怎么进入后台

- 只有被授权的管理员账号，首页“今日推荐”区域才会看到 `进入工作台`
- 普通用户不会看到这个入口
- 就算手动输入后台页面路径，页面和云函数也会再次校验权限
- 如果你还没配置白名单，可以在首页长按日期，直接查看并复制当前账号的 `openid`

## 权限是怎么判断的

- 当前项目使用云函数环境变量 `ADMIN_OPENIDS`
- 变量值填写一个或多个小程序用户 `openid`
- 多个值可以用英文逗号或换行分隔

示例：

```text
oAbc1234567890,oDef0987654321
```

## 需要你做的配置

1. 打开云开发控制台
2. 进入 `cloudfunctions/recipe-manager`
3. 新增环境变量 `ADMIN_OPENIDS`
4. 填入允许进入后台的管理员 openid
5. 重新部署 `recipe-manager`

## 说明

- 微信后台里的“小程序管理员”身份，代码里不能直接自动识别
- 所以这里采用的是更稳定的“openid 白名单”
- 只要把真正的管理员账号 openid 配进去，就能达到“只有管理员看得到后台入口”的效果
