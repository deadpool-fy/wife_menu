# 小程序开发者绑定指南

## 问题现象
```
当前小程序账号未绑定开发者，可前往小程序管理后台绑定开发者
```

## 问题原因
微信小程序的开发者工具账号没有在微信公众平台后台绑定为开发者，导致权限不足，无法使用某些云开发功能。

## 解决步骤

### 第一步：登录微信公众平台
1. 访问 https://mp.weixin.qq.com/
2. 使用小程序的**管理员账号**登录
3. 确认进入了正确的小程序后台

### 第二步：绑定开发者
1. **进入成员管理**：
   - 在左侧菜单找到 **"成员管理"**
   - 或在 **"设置"** → **"基本设置"** → **"成员设置"**

2. **添加开发者**：
   - 点击 **"添加成员"** 或 **"邀请管理员"**
   - 输入您的微信号（用于开发者工具登录的微信号）
   - 选择权限为 **"开发者"** 或 **"管理员"**

3. **接收邀请**：
   - 在微信中接收邀请
   - 点击确认绑定

### 第三步：验证绑定成功
1. **重新登录开发者工具**：
   - 退出当前登录
   - 重新扫码登录

2. **检查云开发权限**：
   - 进入云开发控制台
   - 确认能正常使用所有功能

### 第四步：重新配置权限

1. **云开发权限**：
   - 进入 **"设置"** → **"权限设置"**
   - 开启 **"订阅消息"** 权限

2. **重新部署云函数**：
   - 删除原有云函数（如无法删除，跳过此步）
   - 重新上传部署 `sendMessage` 函数

## 详细操作指南

### 1. 确认小程序信息
在开发者工具中查看：
- **AppID**: `wx5783986edcbaba40`
- **项目名称**: `wife_menu`

### 2. 获取绑定微信号
在开发者工具中：
- 点击右上角用户头像
- 查看登录的微信号
- 这个微信号需要绑定为开发者

### 3. 绑定的微信号权限
建议设置为 **"管理员"** 权限，包含：
- 开发权限
- 云开发权限
- API调用权限

### 4. 常见问题排查

#### Q1: 找不到成员管理入口
- **方式1**: 左侧菜单 → **"成员管理"**
- **方式2**: 顶部导航 → **"设置"** → **"基本设置"** → **"成员管理"**

#### Q2: 无法添加开发者
- 确认是小程序管理员权限
- 小程序必须通过审核
- 部分权限需要企业认证

#### Q3: 接收不到邀请
- 检查微信号是否正确
- 查看微信中的**服务通知**
- 确认微信号已绑定手机号

### 5. 验证步骤

绑定完成后：
1. **重新扫码登录开发者工具**
2. **进入云开发控制台**
3. **测试云函数部署**
4. **检查订阅消息权限**

## 临时解决方案

在开发者绑定完成前：

### 方案1：降级使用
```javascript
// 修改云函数为简单的返回
exports.main = async (event, context) => {
  return {
    success: true,
    message: '开发者绑定完成后可使用完整功能',
    data: event
  }
}
```

### 方案2：本地调试
- 使用云函数本地调试功能
- 避免云端权限问题

### 方案3：复制功能
- 使用降级复制功能
- 手动转发消息

## 权限层级说明

小程序权限层级：
1. **管理员**：完整权限，推荐
2. **开发者**：开发相关权限
3. **体验者**：体验权限，受限
4. **运营者**：运营权限，不含开发

建议直接申请 **"管理员"** 权限。
