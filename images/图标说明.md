# 美食主题图标资源

## 🍽️ 图标说明

本项目使用美食主题的图标来增强视觉效果。以下是已创建的图标资源：

### 📱 TabBar 图标 ✅
- **今日菜单**: `images/menu.png` (40x40px) - 珊瑚红色
- **分类浏览**: `images/category.png` (40x40px) - 珊瑚红色
- **今日菜单-激活**: `images/menu-active.png` (40x40px) - 白色
- **分类浏览-激活**: `images/category-active.png` (40x40px) - 白色

### 🎨 图标设计

#### 今日菜单图标 (menu.png)
- 设计：三条水平线 + 一个圆点
- 颜色：珊瑚红 (#ff6b6b)
- 激活状态：白色，更醒目

#### 分类浏览图标 (category.png)
- 设计：2x2网格布局
- 颜色：珊瑚红 (#ff6b6b)
- 激活状态：白色，更醒目

### 🎯 设计原则

1. **一致性**: 所有图标使用相同的设计风格
2. **简洁性**: 图标简洁明了，易于识别
3. **色彩**: 使用项目主色调 (#ff6b6b)
4. **尺寸**: 符合微信小程序规范 (40x40px)

### 🔧 使用方法

图标文件已放置在 `images/` 目录下，在 `app.json` 中已正确配置：

```json
{
  "tabBar": {
    "list": [
      {
        "pagePath": "pages/index/index",
        "text": "今日菜单",
        "iconPath": "images/menu.png",
        "selectedIconPath": "images/menu-active.png"
      },
      {
        "pagePath": "pages/category/category",
        "text": "分类浏览",
        "iconPath": "images/category.png",
        "selectedIconPath": "images/category-active.png"
      }
    ]
  }
}
```
