# TabBar 图标说明

## 需要的图标文件

为了完善小程序的 tabBar 显示效果，需要以下4个图标文件：

1. `menu.png` - 菜单图标（未选中状态）
2. `menu-active.png` - 菜单图标（选中状态）
3. `category.png` - 分类图标（未选中状态）
4. `category-active.png` - 分类图标（选中状态）

## 图标规格要求

- 尺寸：40x40 像素
- 格式：PNG
- 背景：透明
- 颜色：未选中状态建议使用灰色 (#666666)，选中状态使用主题色 (#ff6b6b)

## 图标获取方式

### 1. 在线图标库
- **Iconfont (阿里巴巴矢量图标库)**: https://www.iconfont.cn/
- **IconPark**: https://iconpark.oceanengine.com/
- **Feather Icons**: https://feathericons.com/

### 2. 推荐图标关键词
- 菜单图标：menu, restaurant, food, dish
- 分类图标：category, grid, list, classification

### 3. 下载步骤
1. 访问图标网站
2. 搜索相关关键词
3. 选择合适的图标
4. 下载 PNG 格式，40x40 像素
5. 重命名为对应文件名
6. 放入此 images 文件夹

### 4. 使用图标后
下载图标后，需要在 `app.json` 中恢复图标配置：

```json
"tabBar": {
  "color": "#666666",
  "selectedColor": "#ff6b6b",
  "backgroundColor": "#ffffff",
  "borderStyle": "black",
  "list": [
    {
      "pagePath": "pages/index/index",
      "text": "今日菜单",
      "iconPath": "images/menu.png",
      "selectedIconPath": "images/menu-active.png"
    },
    {
      "pagePath": "pages/category/category",
      "text": "分类浏览",
      "iconPath": "images/category.png",
      "selectedIconPath": "images/category-active.png"
    }
  ]
}
```
