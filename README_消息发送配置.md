# 消息发送功能配置说明

## 功能概述

小程序已实现多种消息发送方式，可以将菜单、评价等信息发送给指定的用户（您的丈夫）。

## 发送方式

### 1. 发送微信消息
- **功能**：自动复制消息到剪贴板，提示用户发送给指定用户
- **优点**：简单直接，无需额外配置
- **缺点**：需要手动粘贴发送

### 2. 复制到剪贴板
- **功能**：将消息内容复制到手机剪贴板
- **优点**：简单可靠，可手动粘贴发送
- **缺点**：需要手动操作

### 3. 分享给朋友
- **功能**：通过微信分享功能发送
- **优点**：使用微信原生分享
- **缺点**：需要手动选择接收人

## 配置步骤

### 1. 修改目标用户信息

编辑 `config/messageConfig.js` 文件：

```javascript
TARGET_USER: {
  // 替换为您丈夫的实际微信ID
  WECHAT_ID: 'Dingding7654321_',
  // 您丈夫的微信昵称
  NICKNAME: '宝宝',
  // 手机号（可选）
  PHONE: '18056052398'
}
```

### 2. 配置模板消息（可选）

如果您有微信小程序模板消息权限：

```javascript
TEMPLATE_MESSAGE: {
  // 菜单模板ID
  MENU_TEMPLATE_ID: 'your_menu_template_id',
  // 评价模板ID
  RATING_TEMPLATE_ID: 'your_rating_template_id',
  // 菜品推荐模板ID
  DISH_TEMPLATE_ID: 'your_dish_template_id'
}
```

## 使用说明

### 发送菜单
1. 在首页选择菜品
2. 点击"发送给老公"
3. 选择发送方式：
   - 发送微信消息
   - 复制到剪贴板
   - 分享给朋友

### 发送评价
1. 在评价页面完成评分
2. 点击"提交评价"
3. 选择发送方式

### 分享菜品
1. 在菜品详情页
2. 点击"分享菜品"
3. 选择发送方式

## 推荐配置

### 最简单的方式
1. 使用"发送微信消息"功能
2. 自动复制到剪贴板，提示发送给指定用户
3. 无需额外配置

### 备选方式
1. 使用"复制到剪贴板"功能
2. 手动粘贴到微信发送给丈夫
3. 简单可靠

## 注意事项

1. **微信ID获取**：需要您丈夫的微信ID，可以通过微信设置查看
2. **权限配置**：某些功能需要小程序后台配置相应权限
3. **测试建议**：建议先使用"复制到剪贴板"功能测试
4. **隐私保护**：请确保配置信息的安全性

## 故障排除

### 微信消息发送失败
- 检查剪贴板权限是否正常
- 确认消息内容是否过长
- 尝试使用其他发送方式

### 模板消息发送失败
- 检查模板ID是否正确
- 确认用户是否已授权接收模板消息
- 检查小程序是否有模板消息权限

### 复制功能不工作
- 检查手机剪贴板权限
- 尝试重启小程序
- 使用分享功能作为备选方案

## 技术支持

如有问题，请检查：
1. 配置文件是否正确
2. 网络连接是否正常
3. 微信版本是否支持相关功能
